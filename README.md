## Setup âš™ï¸

### Download the project:
```shell
git clone https://github.com/nemo256/graphs
cd graphs
```

### Install the required dependencies:
```shell
yarn install
```
or
```shell
npm install
```

### Launch a development server:
```shell
yarn dev
```
or
```shell
npm run dev
```

### Check localhost port 3000:
```shell
firefox localhost:3000
```
